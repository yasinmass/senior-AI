# SeniorMind AI 🧠

> Voice-first mental health & dementia care 
> platform built for senior citizens.

## What is SeniorMind AI?

SeniorMind AI lets seniors speak naturally 
in their own language. No typing. No complex 
navigation. Just voice.

## Features

🤖 **Mitra** — AI companion that listens and responds  
📔 **Voice Diary** — Speak your thoughts, AI saves them  
💛 **Daily Check-in** — Voice-guided wellness questions  
🎮 **Memory Game** — Cognitive training tracked by doctors  
👨‍⚕️ **Doctor Portal** — Real-time emotional analytics  
🚨 **Crisis Detection** — Auto alerts to caretakers  

## Tech Stack

- **Frontend** — React + Vite
- **Backend** — Django + Python
- **AI Models** — Whisper, LLaMA3, pysentimiento
- **Database** — SQLite3
